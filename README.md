# Toptal calories project

**How to start the project:**

`1. Use node 14.18`

`2. Clone the project`

**Terminal 1:**

`3. cd ./toptal-server`

`4. npm install`

`5. create a .env file, with content: "PORT=3200"`

`6. npm run dev`

**Terminal 2:**

`7. cd ./toptal-client`

`8. npm install`

`9. npm run start`
